# 🤖 HR Policy Explainer Chatbot for Employee Onboarding

This project is a web-based chatbot developed using **Django** to assist new employees during the onboarding process. The chatbot provides quick and accurate answers to common HR-related questions such as leave policies, working hours, salary details, employee benefits, and company guidelines.

The goal of this project is to improve employee onboarding by providing instant support while reducing the workload of HR teams.

---

## 📁 Project Structure

| Folder/File                    | Description                                                   |
| ------------------------------ | ------------------------------------------------------------- |
| `manage.py`                    | Django project management file                                |
| `HR_Chatbot/`                  | Main Django project configuration                             |
| `chatbot/`                     | Chatbot application containing views, URLs, and chatbot logic |
| `templates/chatbot/index.html` | Chatbot user interface                                        |
| `static/css/style.css`         | Styling and responsive design                                 |
| `static/js/script.js`          | Frontend chatbot functionality                                |
| `static/images/chatbot.png`    | Chatbot logo/icon                                             |
| `.env`                         | Stores API keys securely                                      |

## 🔄 Workflow

1. Employee enters a question in the chatbot.
2. The query is sent to the Django backend.
3. The chatbot processes the request.
4. AI API or chatbot logic generates a response.
5. The response is displayed instantly to the employee.

**Flow:**

`User → Chat UI → Django Backend → Chatbot Logic/API → Response → User`

## 🚀 Getting Started

1. Clone the repository.
2. Install the required dependencies.
3. Configure the API key in the `.env` file.
4. Run the Django development server:

   ```bash
   python manage.py runserver
   ```
5. Open your browser and visit:

   ```
   http://127.0.0.1:8000/
   ```
